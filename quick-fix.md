# Quick Fix

## Overview

Diagnose the reported issue's root cause and implement a minimal, effective solution. Prioritize stability.

## Steps

1. **Diagnosis**
    - Analyze logs and trace execution to isolate the root cause.
    - Distinguish between symptom and core problem.
2. **Strategy**
    - Design the simplest effective fix.
    - Ensure adherence to project guidelines.
    - Validate that the fix addresses the root cause.
3. **Execution**
    - Apply the fix cleanly.
    - Verify error resolution.
    - Check for regressions.

## Quick Fix Checklist

- [ ] Root cause identified
- [ ] Simple, effective solution designed
- [ ] Guidelines respected
- [ ] Fix verified
- [ ] No regressions introduced
